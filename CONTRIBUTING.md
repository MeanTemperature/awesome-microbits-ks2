## Contributing

Got a KS2-friendly micro:bit resource to add? Contributions are very welcome.

**Easiest way (no GitHub account needed):**
[Email me](mailto:justinheath33@gmail.com?subject=awesome-microbits-ks2%20suggestion) 
with the details below and I'll add it.

**If you're comfortable with GitHub:**
Submit a [pull request](https://github.com/MeanTemperature/awesome-microbits-ks2/pulls) 
editing the README directly.

### What to include

When suggesting a resource, please provide:

- **Name** - the resource title
- **Link** - direct URL
- **One-sentence description** - what it is and what it does
- **Why it's KS2-appropriate** - age suitability, curriculum link, or how you've used it
- **Suggested section** - where on the list it fits (or suggest a new one)

### What fits

This list is curated for **UK Key Stage 2** (ages 7-11), with a focus on:

- Extension activities for more able primary pupils
- Primary-age code clubs
- Teacher CPD and classroom resources

Resources pitched at secondary, hobbyist electronics, or professional development tend to get filtered out - but if you think something's borderline and valuable, flag it and make the case.
